spfrontier

A set of tools for estimation (MLE) of various spatial specifications of stochastic frontier models

News

  • First version is released (2014-Mar-21)
  • Further news about the spfrontier project are available at the project's R-Forge site.

Supported spatial stochastic frontier specifications

  • Spatial specfifications:
    • Spatil lags of a dependent variable
    • Spatil lags of a symmetric error component
    • Spatil lags of an inefficiency error component
  • INefficiency term distribution specifications
    • Half-normal distribution
    • Truncated normal distribution

Authors

  • Dmitry Pavlyuk

Repositories

  • The released version is available on CRAN
    • You can download the source code and binaries from http://cran.r-project.org/web/packages/spfrontier/index.html.
    • You can install the package by typing at the R console: 
       install.packages( "spfrontier" )
  • The development version is available on R-Forge
    • You can download the source code and binaries from https://r-forge.r-project.org/scm/?group_id=1776.
    • You can install the package by typing at the R console: 
       install.packages( "spfrontier", repos="http://R-Forge.R-project.org" )
